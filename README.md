# 土屋プロトコル - 競馬予想AIシステム

**TsuchiyaProtocol-Omega v7.0** — 物理演算・機械学習による競馬予想WEBアプリ

## 機能

- 🏇 **出馬表登録** - 手動入力 / テキスト貼り付けパーサー（5000文字対応）
- 🛰️ **AI予想実行** - 土屋プロトコル（物理演算・血統・騎手・枠順）
- 🎯 **買い目生成** - 13点精密フォーメーション（3-3-7構造）三連複/馬連/馬単
- 🎯 **WIN5対応** - 5レース選択型予想
- ✅ **結果入力・的中判定** - 自動的中チェック
- 🧬 **自動学習** - 結果に基づく学習パッチ自動生成・蓄積
- 📈 **成績統計** - 的中率・ROI・競馬場別分析

## 対応競馬場

**地方競馬**: 大井 / 門別 / 笠松 / 名古屋 / 弥富 / 他  
**中央競馬**: 阪神 / 中山 / 東京 / 京都 / 他
