# 小倉競馬場 コース特性と戦略的分析ドキュメント

## 1. 核心的な概念：小倉競馬場を支配する「スピード絶対主義」と「高低差」
小倉競馬場は「平坦・小回り」と認識されがちですが、実際には無視できない高低差と、スピードを殺さないための計算された幾何学的構造を持っています。

*   **無視できない高低差**: 芝コースで3.0m、ダートコースで2.9mの高低差が存在します。最高地点は2コーナーにあり、そこから向正面、3〜4コーナーにかけて緩やかな下り坂が続くレイアウトです。
*   **スパイラルカーブの採用**: 3〜4コーナーには入口が緩く出口がきつい「スパイラルカーブ」が採用されており、下り坂でついた勢い（遠心力）を逃がしながら、スピードを落とさずにコーナーを旋回できる設計になっています。
*   **直線距離**: 芝293m、ダート291.3mと短くフラットです。下り坂での慣性をそのまま直線に持ち込める「スピード持続力」が勝負を分けます。
*   **スピード絶対主義**: スタート直後からの下り坂により、特に芝1200mはJRA全10場屈指のハイペース特化コースとなります。

## 2. コース別・バイアス別の特徴と「勝ちパターン」

*   **枠順と馬場バイアス**
    *   **芝**: 開幕週などは「内枠」が圧倒的に有利。開催が進み内が傷むと、特に芝1200mで開催後半（7日目以降）は「外差し」傾向へシフトし、7・8枠の回収率が向上します。
    *   **ダート**: 良馬場では中〜外枠が好走しやすいが、雨で湿ると「内枠・前有利」が顕著になります。
*   **脚質別成績**
    *   直線が短いため「逃げ・先行」が圧倒的有利（芝1200mの逃げ馬勝率は23〜33%）。
    *   中長距離（芝1800m以上、ダ1700m）では、2コーナーの丘でペースが緩むため、向正面からの下り坂を利用した**「捲り（マクリ）」**の期待値が非常に高いです。
*   **優先すべき「勝ちパターン」**
    「テンのスピードで先行し、下り坂とスパイラルカーブの慣性を活かして押し切る機動力勝負」。中距離ではコースロスを恐れず自ら動いて4コーナーで先頭集団に取り付く「捲り」が狙い目です。

## 3. レース階層別コース特性の変化

*   **下級条件（未勝利〜1勝クラス）**
    *   「行き切る絶対スピード」だけで押し切れるケースが多い。
    *   前半3Fが33秒台後半〜34秒台で推移。純粋なダッシュ力を持つ逃げ馬を信頼。
*   **上級条件（重賞・OP）**
    *   前半3Fが32秒台という過酷な超前傾ラップがデフォルト（例：北九州記念）。
    *   極限の激流によりラスト200mで失速する逆転現象が起きるため、ハイペースを追走しながら最後までバテない「持続力・底力」を備えた馬や、好位〜中団から差せる馬に狙いをシフト。

## 4. クラス別「激走」のトリガーと必須条件

小倉のトリッキーな物理条件とタフな馬場は、特定の適性を持つ穴馬を浮上させます。

### ① 激走の客観的指標
*   **馬格（馬格体重）**: 芝1200mでは**馬体重500kg以上の大型馬**の回収率が単複100%超。激しいポジション争いでの当たり強さと、高速ペースを維持する筋肉量（パワー）が直結。
*   **距離短縮**: 前走から距離短縮で臨む馬は回収率90%超。前走の長い距離で培われた「追走体力・スタミナ」が前傾ラップの消耗戦で活きます。
*   **脚質（逃げ馬）**: 特定データで勝率33.3%、単回率396と暴力的な数値を記録。

### ② 血統（種牡馬）における激走条件
*   **芝スプリント**: ロードカナロア、ビッグアーサー等の完成されたスピード血統。冬のタフな馬場では欧州系（ガリレオ系等）が浮上。
*   **ダート1700m**: 
    *   **スマートファルコン**: 揉まれ弱いため**5〜8枠（外枠）に入った際の単回率333**と極大化。
    *   クロフネ、オルフェーヴル（器用さ）、シニスターミニスター。湿ったダートではカジノドライヴやパイロ。

### ③ 騎手と機動力
*   **ローカル巧者**: 幸英明、松山弘平、鮫島克駿といった小回りでの仕掛け処を熟知した騎手。
*   **マクリ実績**: 中距離戦では「前走4角2〜3番手」など、道中で動ける機動力を持つ馬が穴の共通点。

## 5. 小倉競馬場 コース詳細幾何学データと砂の物理特性

### ① 幾何学データ
*   **芝コース（高低差3.0m）**
    *   1周距離: A 1,615.1m / B 1,633.9m / C 1,652.8m
    *   直線 293m / 幅員 30m / コーナー角 113（急）
*   **ダートコース（高低差2.9m）**
    *   1周距離 1,445.4m / 直線 291.3m / コーナー角 86（**全JRA場で最も急**）
    *   1〜2コーナー中間の丘（頂上）から4コーナーまで下り。ゴール前直線は高低差0.6mの上り。

### ② ダートコース「砂の物理特性」
*   **砂のブレンド比率**: 青森県産70%、オーストラリア・アルバニー産25%、宮崎県産5%。
    *   **青森産**: グリップ力が高い（インターロッキング）。水分を含むと高速化。
    *   **オーストラリア産**: 粒子が極めて丸い。透水性（排水能力）に優れ、泥田化を防ぐ。脚元への衝撃を分散するクッション効果。
*   **砂厚**: 標準 85mm。
*   **メンテナンス**: 採砂地から遠いため、洗浄して再利用する運用が中心。これにより粒子が細かくなり、ダートが軽くなって速い時計が出やすくなる特性があります。

## 6. コーナーにおける遠心力ロスとバンクの物理的影響
*   **遠心力の極大化**: 最も急なコーナー角（86）と、下り坂による速度増（$v^2$に比例）により、他場以上に外への物理的負荷が大きい。
*   **バンクとスパイラルカーブの恩恵**: 適切なバンク（傾斜）とスパイラルカーブの組み合わせにより、慣性を殺さず、姿勢制御へのエネルギーロスを最小限に抑えて直線へ向かうことが可能になっています。
*   **路盤の反発力**: エクイターフへの張替えやオーバーシードにより、グリップの効く良好な馬場状態（高い反発力）がスピードの持続を支えています。
