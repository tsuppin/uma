# 笠松競馬予想GEM：クラス編成・能力分析マスターレポート

## 1. 核心的戦略：番組賞金パズルの「歪み」を特定せよ
笠松競馬におけるクラス編成は、獲得賞金と換算率が織りなす**「ロジカルなパズル」**です。実力と格付けの乖離を特定することが、高い回収率への唯一の道となります。

### ① 番組賞金算出と格付けのメカニズム
*   **クラス境界線**: 
    *   C級： 250万円未満
    *   B級： 250万円以上 450万円未満
    *   A級： 450万円以上
*   **他地区からの転入換算率（一例）**:
    *   JRA（2〜4歳）：35%
    *   南関東（2〜4歳）：50%
    *   兵庫・高知：80%
*   **「JRA経験馬への70万円加算」トラップ**:
    JRAで1走でも経験があれば、換算賞金に一律70万円が上乗せされます。これにより賞金0円のJRA未勝利馬が、生え抜きの実力馬が揃うC10組前後に強制配置される「負の期待値」が発生します。

## 2. 「降級馬」の期待値と能力の歪み分析

年度替わりや年齢による賞金カットは、意図的な格差を作り出します。

*   **降級無双の3条件**:
    1.  **鞍上強化**: リーディング1位・渡邉竜也騎手への乗り替わり（勝率30%超）。
    2.  **タイム比較**: B級時代の走破タイムがC級平均を1秒以上上回っているか。
    3.  **馬体重**: 維持または微増（パワーの維持）。
*   **チェックリスト**:
    *   制度降級（前走B級→今走C級）か？
    *   勝負気配（リーディング上位騎手への手替わり）はあるか？

## 3. 他地区転入馬の精密スクリーニング

転入馬は「名前負け」による過剰人気と、「格付けの恩恵」による過小評価を同時発生させます。

*   **期待値の高いパターン**:
    *   **名古屋 → 笠松**: 同一地区。名古屋の重い砂で苦戦していた馬が、笠松の軽い砂と小回りで一変。
    *   **門別(北海道)2歳 → 笠松**: 門別の2歳レベルは突出。下級条件勝ちでも重賞級。
*   **注意すべきパターン**:
    *   **南関C1 → 笠松A級**: 数値上はA級だが、南関の下級条件と笠松の最上位クラスでは質が異なり、過剰人気しやすい。

## 4. クラス別・走破タイムのベンチマーク（1400m）

賞金という政治的数値の歪みを、タイムという物理的数値で修正します。

*   **タイム逆転現象**: 先行争いが激化するB級のタイムが、競り合いの緩いC級を下回ることがあります。
*   **基準値（1400m）**:
    *   A級: 1分27秒5 ～ 1分28秒5
    *   B級: 1分29秒0 ～ 1分30秒0
    *   C級: 1分30秒5 ～ 1分32秒5
*   **判定ロジック**: 前半の競り合いを制しながら1分29秒台を記録したC級馬は「実質B級」と判定。

## 5. GEM 期待値自動判定ロジック（IF/THEN実装用）

1.  **【降級無双】**:
    `IF [前走クラス >= B] AND [今回クラス == C] AND [Jockey == 渡邉竜也] AND [馬体重変動 <= 5kg] THEN [EV: S]`
2.  **【JRA歪み補正】**:
    `IF [JRA未勝利転入] AND [番組賞金額 < 100k] AND [距離 == 800m] THEN [EV: A]`
    (※800mはJRAのスピードが活き、加算金の影響が最小化される)
3.  **【タイムレベル逆転】**:
    `IF [前走タイム <= 当該日の上位クラス平均] AND [前走 1角通過順位 <= 3] THEN [EV: A+]`
